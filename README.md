📊 Retail Customer Behavior Analysis
🧠 Business Problem

A leading retail company aims to understand customer shopping behavior to improve sales, customer satisfaction, and retention. With changing trends across demographics, product categories, and sales channels (online vs. offline), the goal is to identify key factors influencing purchasing decisions such as discounts, reviews, seasons, and payment methods.

Key Question:

How can consumer shopping data be leveraged to identify trends, improve engagement, and optimize marketing and product strategies?

🚀 Project Overview

This project performs end-to-end data analysis using Python, SQL, and Power BI:

Data Preparation (Python): Cleaned and transformed raw data using Pandas
Data Modeling & Analysis (SQL): Structured data, created analytical tables, and extracted insights using complex queries (joins, aggregations, window functions)
Visualization (Power BI): Built interactive dashboards to highlight trends and KPIs
Insights & Recommendations: Derived actionable business insights for decision-making
🛠️ Tech Stack
Python: Pandas, NumPy
SQL: Data Cleaning, Joins, Aggregations, Window Functions
Visualization: Power BI
Database: SQL Server / MySQL

📌 Key Analysis Performed

Customer segmentation based on purchasing behavior
Impact of discounts on product sales
Category-wise and product-wise performance analysis
Identification of top-selling products and trends
Analysis of purchase patterns across channels and demographics

📈 Key Insights

Discounts significantly influence purchase decisions for certain product categories
Specific products show higher repeat purchase behavior
Customer preferences vary across demographics and channels
Seasonal trends impact buying pattern
